# 差分数组 (Difference Array) 全家桶速查表

差分的核心思想是**前缀和的逆运算**，主要用于将 $$O(N)$$ 或 $$O(N^2)$$ 的区间修改操作降维打击为 $$O(1)$$ 的端点修改。

---

## 1. 一维差分 (1D Difference)
**使用场景**：频繁对一维数组的区间 $$[L, R]$$ 加上相同的值 $$v$$。

*   **定义**：$$D[i] = A[i] - A[i-1]$$
*   **区间修改** ($$O(1)$$):
    *   $$D[L] \mathrel{+}= v$$ （起点加上 $$v$$）
    *   $$D[R+1] \mathrel{-}= v$$ （终点后一个位置减去 $$v$$，抵消影响）
*   **还原数组** ($$O(N)$$): 对差分数组求前缀和。
    *   $$A[i] = A[i-1] + D[i]$$

---

## 2. 二维差分 (2D Difference)
**使用场景**：频繁对二维矩阵中左上角为 $$(x_1, y_1)$$，右下角为 $$(x_2, y_2)$$ 的子矩阵加上相同的值 $$v$$。

*   **定义** (容斥原理)：$$D[i][j] = A[i][j] - A[i-1][j] - A[i][j-1] + A[i-1][j-1]$$
*   **区间修改** ($$O(1)$$):
    1.  $$D[x_1][y_1] \mathrel{+}= v$$ （左上角加）
    2.  $$D[x_1][y_2+1] \mathrel{-}= v$$ （右上角外侧减）
    3.  $$D[x_2+1][y_1] \mathrel{-}= v$$ （左下角外侧减）
    4.  $$D[x_2+1][y_2+1] \mathrel{+}= v$$ （右下角外侧加回来）
*   **还原数组** ($$O(N \times M)$$): 对差分数组求二维前缀和。
    *   $$A[i][j] = D[i][j] + A[i-1][j] + A[i][j-1] - A[i-1][j-1]$$

---

## 3. 树上差分 (Tree Difference)
**使用场景**：在树形结构中，频繁对节点 $$u$$ 到节点 $$v$$ 的最短路径上的所有节点（或边）加上相同的值 $$x$$。需要结合 **LCA (最近公共祖先)** 算法。

### 3.1 点差分 (修改路径上的节点)
*   **操作**：
    1.  $$D[u] \mathrel{+}= x$$
    2.  $$D[v] \mathrel{+}= x$$
    3.  $$D[\text{lca}(u, v)] \mathrel{-}= x$$
    4.  $$D[\text{parent}(\text{lca}(u, v))] \mathrel{-}= x$$
*   **还原**：自底向上 DFS，每个节点的最终值等于其**子树中所有节点的差分值之和**。

### 3.2 边差分 (修改路径上的边)
*(注：通常将边的值存入其深度较深的下端点节点中)*
*   **操作**：
    1.  $$D[u] \mathrel{+}= x$$
    2.  $$D[v] \mathrel{+}= x$$
    3.  $$D[\text{lca}(u, v)] \mathrel{-}= 2x$$
*   **还原**：同上，自底向上 DFS 求子树和。

---

## 4. 进阶：树状数组 + 差分 (Fenwick Tree)
**使用场景**：既要**频繁区间修改**，又要**频繁单点/区间查询**（在线操作）。普通差分只能在最后统一查询（离线操作）。

*   **核心思路**：用树状数组来维护**差分数组** $$D$$。
*   **区间修改**：在树状数组中执行 `add(L, v)` 和 `add(R+1, -v)`，时间复杂度 $$O(\log N)$$。
*   **单点查询**：查询原数组 $$A[x]$$ 的值，等价于求差分数组的前缀和，即在树状数组中执行 `query(x)`，时间复杂度 $$O(\log N)$$。
