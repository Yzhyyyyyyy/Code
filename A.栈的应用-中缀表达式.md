# 双栈法表达式求值：核心心法与避坑指南

在处理数学表达式求值（如计算器、24点游戏）时，“双栈法”是最经典且高效的算法。本文档总结了双栈法的核心逻辑及常见致命易错点。

## 一、 核心数据结构
我们需要维护两个栈：
1. **数字栈 (`num`)**：存放操作数。
2. **符号栈 (`fuhao`)**：存放运算符。

**基本动作（弹栈计算三步曲）：**
每次执行计算时，严格执行以下三步：
1. 从符号栈弹出一个运算符 `op`。
2. 从数字栈弹出**右操作数** `b`（注意：先弹出的是右边的数！）。
3. 从数字栈弹出**左操作数** `a`。
4. 计算 $$a \text{ op } b$$，并将结果压回数字栈。

---

## 二、 核心心法：什么时候该弹栈？

双栈法的灵魂在于**运算符优先级比较**。当我们遍历表达式，遇到一个新的运算符时，必须与符号栈的**栈顶运算符**进行优先级比较。

### 1. 同级运算必须“从左到右”计算（为什么是 `>=`）
**规则**：只要栈顶符号的优先级 **大于或等于** 当前符号，就必须弹出计算！

**反面教材**：如果不包含“等于”，遇到同级符号直接压栈，会导致计算顺序从右向左。
例如计算 $$9 - 9 + 8$$：
* 如果 `+` 直接压在 `-` 上面，最后清空栈时，会先算 $$9 + 8 = 17$$，再算 $$9 - 17 = -8$$，**结果全错**！
* **正确做法**：遇到 `+` 时，发现栈顶是同级的 `-`，触发 `>=` 条件。先把前面的 $$9 - 9 = 0$$ 算完，再把 `+` 压栈，最终计算 $$0 + 8 = 8$$。

### 2. 为什么必须用 `while` 循环？（推土机效应）
**规则**：不能只用 `if` 弹一次，必须用 `while` 循环一直弹，直到栈空或栈顶优先级严格小于当前符号。

**场景还原**：计算 $$2 + 3 \times 4 - 5$$
* 当读到 `-` 时，此时符号栈里是 `[+, x]`（加号在底，乘号在顶）。
* `-` 的优先级低于 `x`，也等于 `+`。
* 如果只用 `if` 弹一次，只会把 `x` 弹出来算掉，导致 `-` 直接压在 `+` 上面，变成 `[+, -]`，最后计算顺序又乱了。
* **正确做法**：用 `while` 循环。第一轮弹掉 `x`（算 $$3 \times 4 = 12$$），第二轮继续判断发现栈顶是 `+`，优先级等于 `-`，继续弹掉 `+`（算 $$2 + 12 = 14$$）。就像推土机一样，把前面所有优先级 `>=` 它的统统推平！

## 三、 语言与题目特性避坑（C++特供）

### 1. 负数除法的“向下取整”
题目要求除法为数学意义上的向下取整（例如 $$-3 / 7 = -1$$）。
**坑点**：C++ 中的整数除法 `/` 是**向零取整**（$$-3 / 7$$ 在 C++ 中结果为 $$0$$）。
**解法**：必须引入 `<cmath>` 库，使用 `floor((double)a / b)` 来强制向下取整。

### 2. 字符与数字的转换
如果是固定格式的字符串（如 `a+bxc/d`），提取数字时记得减去字符 `'0'`：`a[i] - '0'`。

## 四、 进阶：如何优雅地处理括号？

遇到带括号的表达式，有两种主流处理方式：

### 思路一：递归下降法（分治思想）
* **逻辑**：遇到左括号 `(` 时，寻找匹配的右括号 `)`，将括号内的子串提取出来，当作一个全新的表达式，递归调用求值函数。
* **特点**：符合人类直觉，但需要额外写代码来匹配括号和截取字符串。

### 思路二：栈内“筑墙”法（双栈原生支持，极其优雅）
不需要递归，只需要给括号设定特殊的优先级规则：
1. **左括号 `(` 优先级设为最低（如 0）**。
   * 遇到 `(` 无脑压入符号栈。它就像一堵墙，因为优先级最低，后面的 `+ - * /` 都无法把它弹出来，只能乖乖压在它上面。
2. **遇到右括号 `)` 时，触发“局部清算”**。
   * 疯狂弹栈并计算，**直到符号栈的栈顶是左括号 `(` 为止**。
   * 算完后，把 `(` 弹出并丢弃（拆墙）。

## 五、 终极代码模板 (C++)

```cpp
#include <bits/stdc++.h>
using namespace std;

// 定义优先级
int get_pri(char c) {
    if (c == 'x' || c == '*') return 2; // 乘除最高
    if (c == '/') return 2;
    if (c == '+' || c == '-') return 1; // 加减其次
    if (c == '(') return 0;             // 左括号垫底（筑墙法）
    return -1;
}

// 核心计算逻辑
int calculate(int b, int a, char c) {
    if (c == '+') return a + b;
    if (c == '-') return a - b;
    if (c == 'x' || c == '*') return a * b;
    if (c == '/') return floor((double)a / b); // 修复C++向零取整的问题
    return 0;
}

int evaluate(string s) {
    stack<int> num;
    stack<char> fuhao;
    
    for (int i = 0; i < s.length(); i++) {
        char c = s[i];
        
        if (isdigit(c)) {
            // 处理多位数字的情况（如果是单数字直接 num.push(c-'0') 即可）
            int val = 0;
            while (i < s.length() && isdigit(s[i])) {
                val = val * 10 + (s[i] - '0');
                i++;
            }
            i--; // 抵消 for 循环的 i++
            num.push(val);
        } 
        else if (c == '(') {
            fuhao.push(c); // 筑墙
        } 
        else if (c == ')') {
            // 局部清算，直到遇到左括号
            while (fuhao.top() != '(') {
                char op = fuhao.top(); fuhao.pop();
                int b = num.top(); num.pop();
                int a = num.top(); num.pop();
                num.push(calculate(b, a, op));
            }
            fuhao.pop(); // 拆墙（丢弃左括号）
        } 
        else {
            // 遇到普通运算符：推土机逻辑
            while (!fuhao.empty() && get_pri(fuhao.top()) >= get_pri(c)) {
                char op = fuhao.top(); fuhao.pop();
                int b = num.top(); num.pop();
                int a = num.top(); num.pop();
                num.push(calculate(b, a, op));
            }
            fuhao.push(c); // 当前符号安稳入栈
        }
    }
    
    // 表达式遍历完后，清空栈内剩余的运算
    while (!fuhao.empty()) {
        char op = fuhao.top(); fuhao.pop();
        int b = num.top(); num.pop();
        int a = num.top(); num.pop();
        num.push(calculate(b, a, op));
    }
    
    return num.top(); // 栈顶即为最终结果
}
```
