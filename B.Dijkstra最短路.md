# 🚀 Dijkstra 算法核心笔记

## 一、 算法适用场景
*   **解决问题**：单源最短路径（求从一个固定的起点，到其他所有点的最短距离）。
*   **限制条件**：**边权必须非负**（图中不能有负数长度的边）。

## 二、 核心思想：带有优先队列的 BFS
普通的 BFS（广度优先搜索）是一圈一圈往外找，先找到的步数一定最少。但在带权图中，**“步数少”不等于“总花费少”**（比如走 1 步花费 100，走 3 步花费却只要 15）。

因此，Dijkstra 对 BFS 做了两大升级：
1.  **计算累计路权**：不再只关心“能不能到”，而是时刻记录“到达这里的总花费是多少”。
2.  **引入优先队列（小根堆）**：每次不再盲目扩展周围的点，而是**从当前所有已知路径中，挑一个离起点最近（总花费最小）的点**，以它为跳板去探索别人。

## 三、 核心概念与“装备”
在执行算法时，我们需要准备四个核心工具：

1.  **`dis` 数组（距离表）**：记录起点到每个点的当前最短距离。一开始除了起点是 0，其他全是无穷大（INF）。
2.  **`vis` 数组（确定标记）**：记录哪些点的最短路已经**彻底确定**了。一旦被标记，以后就不再处理它。
3.  **`pq` 优先队列（候选池）**：里面存放着 `(当前距离, 点的编号)`。它会自动把距离最小的点排在最前面。
4.  **`pre` 数组（前驱记录，可选）**：用来记录“我是从哪个点走过来的”，专门用于最后倒推完整的路线。

## 四、 算法完整运行流程

整个算法就像是一个不断“发现新大陆”并“更新地图”的过程，具体分为以下几步：

### 第 1 步：初始化
把所有点到起点的距离设为无穷大，起点的距离设为 0。把起点放入优先队列中。

### 第 2 步：寻找最近的跳板
从优先队列中拿出一个距离起点**最近**的点（假设叫 `u`）。
*   如果这个点 `u` 已经被标记为“已确定”（`vis` 为 true），说明这是个过期的历史记录，直接扔掉不管。
*   如果没有被标记，就把它标记为“已确定”。**此时，起点到 `u` 的最短距离就真正定下来了，以后绝不会再变！**

### 第 3 步：松弛操作（核心灵魂！）
站在点 `u` 上，环顾它所有的邻居（假设其中一个邻居叫 `v`，从 `u` 到 `v` 的路程是 `w`）。
我们在心里算一笔账：
*   **之前的记录**：去 `v` 需要 100 分钟（`dis[v] = 100`）。
*   **现在的发现**：如果我先走到 `u`，再从 `u` 走到 `v`，只需要 `10 + 5 = 15` 分钟（`dis[u] + w = 15`）。

**15 < 100！发现了一条更近的路！**
此时我们做三件事：
1.  **更新距离**：把去 `v` 的距离表更新为 15（`dis[v] = 15`）。
2.  **加入队列**：把这个新发现 `(15, v)` 扔进优先队列里，让 `v` 以后也有机会作为跳板去更新别人。
3.  **记录前驱**：在 `pre` 数组里记下 `pre[v] = u`（告诉 `v`，你是通过 `u` 走过来的，不是飞过来的）。

### 第 4 步：循环往复
重复第 2 步和第 3 步，直到优先队列空了为止。

## 五、 完整 C++ 模板代码

```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <cstring>

using namespace std;

// 定义无穷大，0x3f3f3f3f 是一个很大的数，且两数相加不会溢出 int
const int INF = 0x3f3f3f3f; 

// 边的结构体
struct Edge {
    int to;     // 目标点
    int weight; // 边权
};

const int MAXN = 100005; 
vector<Edge> adj[MAXN]; // 邻接表存图

int dis[MAXN];  // 存储起点到 i 的最短距离
bool vis[MAXN]; // 标记点 i 是否已经彻底确定最短路
int pre[MAXN];  // 记录前驱节点，用于输出路径

// s: 起点, n: 点的总数
void dijkstra(int s, int n) {
    // 1. 初始化
    for(int i = 1; i <= n; i++) {
        dis[i] = INF;
        vis[i] = false;
        pre[i] = -1; // -1 表示没有前驱
    }
    dis[s] = 0;
    
    // 优先队列，存 pair<距离, 点编号>，小根堆
    priority_queue<pair<int, int>, vector<pair<int, int>>, greater<pair<int, int>>> pq;
    pq.push({0, s});
    
    while (!pq.empty()) {
        // 取出当前距离起点最近的点
        int u = pq.top().second;
        pq.pop();
        
        // 如果这个点已经处理过，跳过（懒惰删除）
        if (vis[u]) continue;
        vis[u] = true; 
        
        // 遍历 u 的所有邻居
        for (auto& edge : adj[u]) {
            int v = edge.to;
            int w = edge.weight;
            
            // 松弛操作：如果通过 u 到 v 比直接去 v 更近
            if (dis[v] > dis[u] + w) {  
                dis[v] = dis[u] + w;    
                pre[v] = u;             // 记录：我是从 u 走过来的
                pq.push({dis[v], v});   // 把新发现放入队列
            }
        }
    }
}

// 打印路径函数（递归法）
// 调用 print_path(终点) 即可
void print_path(int t) {
    if (t == -1) return; // 到了起点的前面，结束
    print_path(pre[t]);  // 先打印前面的
    cout << t << " ";    // 再打印自己
}

int main() {
    int n, m, s; // n点数, m边数, s起点
    cin >> n >> m >> s;
    for (int i = 0; i < m; i++) {
        int u, v, w;
        cin >> u >> v >> w;
        adj[u].push_back({v, w});
        // 如果是无向图，加上下面这句：
        // adj[v].push_back({u, w});
    }
    
    dijkstra(s, n);
    
    // 输出到各点的距离
    for (int i = 1; i <= n; i++) {
        if (dis[i] == INF) cout << "INF ";
        else cout << dis[i] << " ";
    }
    return 0;
}
```
